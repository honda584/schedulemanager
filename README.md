# schedulemanager
プロジェクトタイトル
ScheduleManager

概要説明
・予定管理カレンダー
・時間と予定情報を管理できるアプリ

使用技術

・フロントエンド
HTML,CSS,Javasprict
・バックエンド
java
Spring Boot Dev Tools
Spring Web
Spring Data JPA
Thymeleaf
・インフラ
H2 Database
・開発環境
Eclips IDE
GitHub
Windows10
・ツール
Claude 3.7 Sonnet

機能一覧
1カレンダー表示（インデックスページ）
2日付選択で予定登録ページへ移動
3予定登録と確認表示
4カレンダーでの予定表示
5予定詳細表示
6予定キャンセル（削除）機能
